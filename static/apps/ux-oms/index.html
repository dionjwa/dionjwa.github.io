Technical Architecture

How Metapages serves molecular science

The metapage architecture maps directly to OMSF's needs: isolated components (websites and connected websites) for security, typed data pipes for interoperability, and URL-portability for open science. It's not a big complicated framework, but a small module that simply connects editable websites, and these components are standalone (pick and choose).

It also directly integrates with AI tools, allowing for the generation of code from natural language descriptions, and the ability to edit the code in the browser.
